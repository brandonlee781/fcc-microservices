API Project - Image Search Microservice

User Stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example:

https://bl781-ms.herokuapp.com/imagesearch/star wars

Adding ?offset= and a number to the end of the url will paginate through results

Output:

    {
      image: "http://i.imgur.com/aOenemZ.gif",
      text: "Because Star Wars day is tomorrow....",
      source: "https://www.imgur.com/gallery/aOenemZ"
    },
    ...

Example:

https://bl781-ms.herokuapps.com/latest/imagesearch

Output:
